## ðŸš€ Project Overview

Early detection of diabetes can help individuals take timely action and prevent serious health complications.  
This project uses the **Pima Indians Diabetes Dataset** to train a classification model (Logistic Regression) that predicts diabetes risk.

The project includes:

- Data loading & preprocessing  
- Feature scaling  
- Model training & evaluation  
- Saving trained model as `.pkl`  
- A CLI-based prediction system (`predict.py`)  
- Organized, modular project structure  

This project fulfills the structure and documentation requirements given by VIT faculty.
